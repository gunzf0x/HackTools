# HTB VariaType


Script for [HTB VariaType](https://www.hackthebox.com/machines/variatype) to create valid `.designspace` files. Other scripts to solve the box might be released after this box is retired.

## Requirements
Install `fonttools` for Python:
```shell
python3 -m venv venv_fonttools # Create a virtual environment

source venv_fonttools/bin/activate # Activate the environment

pip3 install fonttools # Install the library in the virtual environment
```
